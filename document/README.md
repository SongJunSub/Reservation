# 기술 문서 디렉토리

이 디렉토리는 프로젝트에서 사용하는 각 기술에 대한 학습 및 참고용 문서를 포함합니다.

## 디렉토리 구조

### 📁 spring-webflux
- Spring WebFlux 관련 개념 및 실습 코드
- 리액티브 프로그래밍 패턴
- WebFlux vs Spring MVC 비교

### 📁 jpa-querydsl  
- JPA 엔티티 및 관계 설정
- QueryDSL 활용법 및 복잡한 쿼리 작성
- 블로킹 I/O 처리 패턴

### 📁 r2dbc
- R2DBC 리액티브 데이터베이스 접근
- 논블로킹 I/O 패턴
- JPA vs R2DBC 성능 및 코드 비교

### 📁 kafka
- Kafka 메시징 패턴
- 이벤트 기반 아키텍처
- Producer/Consumer 구현

### 📁 redis
- Redis 캐싱 전략
- 리액티브 Redis 사용법
- 성능 최적화 기법

### 📁 kotlin-coroutines
- Kotlin 코루틴 vs Java Virtual Threads
- 비동기 프로그래밍 패턴 비교
- Reactor와 코루틴 통합

### 📁 architecture
- Hexagonal Architecture 구현
- Clean Architecture 원칙
- 레이어 분리 및 의존성 관리

## 학습 목표
각 기술의 이론적 배경과 실무에서의 활용법을 코드와 함께 문서화하여 향후 참고할 수 있도록 합니다.
